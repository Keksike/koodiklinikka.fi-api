### Start developing
* Install node.js
```
$ git clone git@github.com:koodiklinikka/koodiklinikka.fi-api.git
$ cd koodiklinikka.fi-api
$ npm install
```

* ``` $ npm start ```

Now app will be running on `localhost:9000`.
